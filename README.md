# c4 - Winter 13/14 Compiler Course Final Project
## Compiles a Subset of C

### Running the Compiler:

Usage: ``./build/default/c4 [options] [filename or - for stdin]``  

Options:  
 ``--tokenize``  
 ``--parse``  
 ``--print-ast``  
 ``--compile``  
 ``--optimize``  
 
 Defaults to ``--compile``
 
### Running the Tests
 
 ``make test_lexer``  
 ``make test_parser``  
 ``make test_printer``  
 ``make test_compiler``
